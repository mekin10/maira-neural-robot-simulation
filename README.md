# MAIRA Neural Robot Simulation

Welcome to the **MAIRA Neural Robot Simulation** repository. This project focuses on the modeling, simulation, and control of the MAIRA robot system. It includes comprehensive mathematical models and programmatic implementations for both the kinematics and the dynamics of the robotic arm.

## 👥 Project Team

* **Mekin Mohammed** - *Project Leader & Lead Developer*


## 📑 Project Overview

This repository contains the source code and technical documentation for our group's robotics project. We have developed modular scripts to calculate forward and inverse kinematics, as well as the dynamic properties and control schemes necessary to simulate the MAIRA robot's behavior. 

The project features cross-platform implementations, providing solutions in both **Python** and **MATLAB**.

## 📂 Repository Contents

* **`Dynamics and Control Part.py`**: Python implementation of the robot's dynamic equations and control algorithms.
* **`Dynamics and Control.m`**: MATLAB script for modeling the system dynamics and simulating the control loops.
* **`Kinematics Part.m`**: MATLAB script for computing the forward and inverse kinematics of the MAIRA robot.
* **`Kinematics Part.py`**: Python implementation of the kinematic models.
* **`Mekin Mohammed Controller(S...).pdf`**: Specific documentation and design details regarding the controller designed by the team leader.
* **`Sec_B_Maira_Neura_Robot.pdf`**: The main project report and technical documentation for Section B.

## 🛠️ Technologies Used

* **Python 3.x** (Requires `numpy`, `scipy`, and standard robotics libraries)
* **MATLAB** (Requires the Robotics System Toolbox)
